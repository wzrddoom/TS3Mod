# TS3Mod

TS3Mod is a .NET Framework 4.8 solution developed in Visual Studio. This repository contains the TS3Mod solution and source files for building and extending the project.

## Prerequisites

- Windows
- .NET Framework 4.8 Developer Pack
- Visual Studio (2019, 2022, or newer) or any IDE that supports .NET Framework 4.8

## Getting started

1. Open the solution: TS3Mod.slnx in Visual Studio.
2. Restore NuGet packages (Visual Studio will normally do this automatically).
3. Build the solution (Build -> Build Solution).
4. Run or debug using Visual Studio (F5) or run the produced executable from the build output folder.

Important source files:

- TS3Mod/CoreModule.cs — main module
